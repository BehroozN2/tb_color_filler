Make Your Bot

  • Your function is called each turn to get your move for that turn.
  • You must return one of these values: top, bottom, left, right.
  • Invalid values or values which result in an out of bound position are ignored.
  • Function parameters:
    • x: Your current horizontal position, a number between 1 (left) and 20 (right).
    • y: Your current vertical position, a number between 1 (top) and 20 (bottom).
    • turn: Current turn, a number between 1 and 1000.
    • memory: A JavaScript object which you can use to save values as properties on it. You can access previously stored properties on this object in future turns. Example: 
      if (turn === 1) {
    memory.something = 'abcd';
} else {
    console.log(memory.something);
    // this outputs 'abcd'
    // in browser's console
}
